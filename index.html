2011-m1s2-ter by jsmaniac

Reconception du jeu PtiClic sous Android

Ce projet est une implémentation du jeu PtiClic sous la plate-forme mobile Android. Dans PtiClic, on demande aux joueurs de classer des mots par rapport à un autre dans des catégories telles que "synonyme", "peut contenir", "peut faire" etc.

Install

Il n'est pour l'instant pas possible d'installer ce jeu sur un téléphonne (d'ici quelques semaines cela sera possible).
