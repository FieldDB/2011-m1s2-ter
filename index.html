PtiClic

Reconception du jeu PtiClic sous Android

Description

Dans PtiClic, on demande aux joueurs de classer des mots par rapport à un autre dans des catégories telles que "synonyme", "peut contenir", "peut faire" etc.

Télécharger

Il n'est pour l'instant pas possible d'installer ce jeu sur un téléphonne (d'ici quelques semaines cela sera possible).

Les sources sont disponibles sur github.
